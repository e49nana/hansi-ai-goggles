    • Five AI Modes

    Each mode transforms the goggles into a specialized AI tool.

    SCOUT
    Real-time object and text identification. OCR, technical component ID, scene analysis. GPT-4o Vision processes camera frames every 2 seconds.
    BABEL
    Instant translation from 12+ languages. Image OCR, voice input, or manual text. Auto-detects source language. TTS reads the translation aloud.
    TITAN
    Voice conversational AI with RAG memory. Recalls past observations via vector similarity search. Push-to-talk or hands-free VAD. "What did I see earlier?"
    SURVEY
    Automatic life-logging camera. Captures and AI-describes surroundings every 10 seconds. Builds a searchable, timestamped memory archive.
    COMBAT
    Navigation mode with GPS waypoints and landmark detection. HUD displays directional cues and distance overlays. (Experimental)
